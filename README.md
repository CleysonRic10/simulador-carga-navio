# ⚓ Simulador Naval (Carga e Logística)

## 📍 Contexto real
Operações navais envolvem alta complexidade na gestão de:
- carregamento e descarregamento
- distribuição de peso
- sincronismo entre porão de carga com tanques de lastro
- eficiência logística

Pequenos erros podem impactar diretamente custo, tempo e segurança.

---

## 💡 O que este projeto faz
Este projeto simula operações navais relacionadas à **movimentação entre porão de carga com tanques de lastro**, permitindo visualizar como diferentes cenários afetam o processo.

👉 A proposta é transformar uma operação complexa em uma simulação visual e compreensível.

---

## 🎯 Objetivo profissional
Este projeto foi desenvolvido com foco em:

- Representar digitalmente processos logísticos reais
- Aplicar lógica computacional em cenários operacionais
- Explorar visualização como ferramenta de análise
- Demonstrar capacidade de transformar operação em solução tecnológica
